This is a hero section...

Made with Mnma

It's dead simple. Mnma will automatically make the first < section > tag on your page into a hero section. want a Call to Action button? Just put a button anywhere in this < section > and Mnma will make it a big, prominent CTA button like the one you see below.

It's dead simple to place images here, to. Just put an < img > anywhere in this < section > tag, and Mnma will position and size it properly.

Typography

Mnma provides a nice-looking typescale and font pairing out of the box.

By default, the first h1 inside a < section > tag will get a small underline element underneath it. Just 'cause it looks neat.

< p > elements have a limited width to increase readability.

2. Navbar

The first < nav > element on your page will become a navbar. The navbar at the top of this page is an example.

The navbar can to hold a logo, a desktop nav menu, and a mobile nav. The logo or wordmark can be any element, and your menu should be a < ul >

Nested nav menus

To create a nested nav menu, just place a < ul > inside of one of an < li > in your nav. Refer to the code snippets below for a working example.

Mobile menu

To create a mobile menu, put an < a > tag inside of your nav with a < ul > inside of it You can build your mobile menu inside of that < ul >. See the code snippet below for a working example.

Navbar code snippets

Below is the markup for the navbar at the top of this page

            
                <nav>
        <span><strong>Logo</strong></span>
        <ul>
            <li>Nav item</li>
            <li>Another item
                <ul>
                    <li>A nested item</li>
                    <li>Wouldn't you know it, another item. This one has a very long name.</li>
                </ul>
            </li>
            <li>Yet another</li>
            <li>A final item</li>
        </ul>
        <a>
            <ul>
                <li>Nav item</li>
                <li>Another item
                    <ul>
                        <li>A nested item</li>
                        <li>Wouldn't you know it, another nested item. This one has a very long name.</li>
                    </ul>
                </li>
                <li>Yet another</li>
                <li>A final item</li>
            </ul>
        </a>
    </nav>

3. Figures

You can create a round image with a caption like the one you see below using a < figure > element with an < img > and < figcaption > inside it

Figure code snippet:
          
            <figure>
            <img src="./assets/pic1.jpg" alt="">
            <figcaption>
              Here I am! This is a picture of me. My mom took it. Don't I look nice? I knew you'd say yes ^_^. I'm smiling
              because I know you're going to enjoy using Mnma. Now go build something neat! 
            </figcaption>
          </figure>
