Mnma: the CSS framework that figures out what you want...

based on your markup alone

This hero section was built with markup alone. In fact, this whole page was built with markup alone. No classes, no ID's. You don't have to remember complex HTML structures. Just write HTML and let Mnma figure out the rest.

What's this frameowrk for?

Mnma. is a framework for building simple landing pages without worrying about classes and ID's. Most product landing pages are fairly similar. They have a nav bar, a hero section with an image, cards, a form, and some explanatory text. Mnma has distilled those characteristics into a set of general CSS rulesets that let you write semantic markup and have it look great right outta the box.

Who might use this?

Folks who just wanna deploy

If you're putting your all into a project and just don't have the time or energy to think about a landing page, then Mnma is for you. Mnma will help you build a simple but effective website without much effort.
